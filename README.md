# 💼 Smart Finance Manager

A Java-based console application that simulates a basic banking system with account management and transaction tracking.

---

## 🚀 Features

* Create Account
* Secure Login (PIN-based)
* Deposit & Withdraw Money
* Transfer Money Between Accounts
* Transaction History Tracking
* Data Persistence using File Handling

---

## 🛠️ Technologies Used

* Java (Core + OOP)
* File Handling
* HashMap
* ArrayList

---

## 📂 Project Structure

* `Main.java` → Application entry point & menu
* `User.java` → User data and operations
* `Transaction.java` → Transaction records
* `FileHandler.java` → Save & load data

---

## ▶️ How to Run

```bash
javac *.java
java Main
```
